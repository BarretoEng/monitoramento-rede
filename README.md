# Sistema de Monitoramento com Prometheus, Grafana e Blackbox Exporter

Este projeto configura um ambiente de monitoramento completo com:

- Prometheus (coleta de métricas)
- Blackbox Exporter (ping e testes HTTP)
- Grafana (visualização)

## Requisitos

- Docker
- Docker Compose

## Como subir

```bash
docker-compose up -d
